Anthropic 最新動向 (2024年)

1. 東京オフィス開設

Anthropicは2024年に東京オフィスを開設し、日本での存在感を高めています。この取り組みは、国際的なAI安全研究所との協力関係の一環として位置づけられています。

2. AI安全性への取り組み

Anthropicは、以下の国際機関と協力関係を構築しています：

  • 米国のAI標準革新センター（CAISI）
  • 英国のAI安全研究所
これらの機関と共同で、Claude 3.5 Sonnetの評価を実施するなど、AIの安全性向上に注力しています。

3. Claude 3.5 Sonnetのリリース

2024年6月に、Claude 3シリーズの改良モデルである「Claude 3.5 Sonnet」をリリースしました。主な特徴：

  • PDF処理APIのサポート
  • 高度な言語処理能力
  • 新たな技術的改善

4. サイバーセキュリティへの対応

Anthropicは、AIテクノロジーが悪用される潜在的なリスクに対して積極的に警告を発しています。特に、国家系のハッカーがAIツールを悪用してサイバー攻撃を自動化する可能性について注意を促しています。
